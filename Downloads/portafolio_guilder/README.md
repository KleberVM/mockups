Portafolio - Guilder Paredes Lovera
===================================

Archivos incluidos:
- index.html  -> Página principal (HTML)
- styles.css  -> Estilos (CSS)
- app.js      -> Interacciones y lógica (JavaScript)
- README.md   -> Este archivo

Características:
- Modo oscuro profesional con gradientes sutiles y sombras.
- Cursor personalizado y halo.
- Filtros por tipo de proyecto (Todos / Web / Mobile / Ecommerce / MVP).
- Tarjetas 3D con efecto tilt al mover el mouse (CSS + JS).
- Modal para ver detalles de proyecto.
- Formularios con fallback `mailto:` y descarga de CV (placeholder).
- Diseño responsivo y optimizado para presentación a clientes.

Qué editar rápidamente:
- Reemplaza `guilder@example.com` en index.html y app.js por tu email real.
- Añade capturas en los elementos `.p-thumb` como `background-image:url('ruta/imagen.jpg')`.
- Sustituye los enlaces de demo/repo/Redes por tus URLs reales.

Despliegue:
- Puedes subir estos archivos a Netlify, Vercel (static), GitHub Pages o tu hosting preferido.
- Si quieres, convierto esto a React + Tailwind y te preparo el repo listo para deploy.

Si quieres cambios (ej.: WebGL, animaciones más avanzadas con THREE.js, integración de formularios con backend, o una versión en React), dime cuál prefieres y lo preparo.
